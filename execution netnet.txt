## 3. CORE ENGINE (THIS IS THE HEART)

### 3.1 Render Loop (GAME LOOP)

```ts
export function animate(time: number) {
  requestAnimationFrame(animate);

  World.update(time);
  CameraSystem.update();
  InteractionSystem.update();

  renderer.render(scene, camera);
}
```

✔ Always running
✔ Always updating
✔ Nothing static

### 3.2 World State Engine

```ts
World = {
  trafficLevel: 0.4,
  timeOfDay: 0.5,
  alerts: [],
  update(dt) {
    this.trafficLevel += Math.sin(dt * 0.001) * 0.01;
  }
}
```

Everything references **World State**
→ guarantees realism and causality.

## 6. INTERACTIVE OBJECT SYSTEM (REAL)

### Router Object

```ts
class Router {
  traffic = Math.random();

  update(dt) {
    this.traffic += Math.sin(dt * 0.002) * 0.01;
    this.led.material.emissiveIntensity = this.traffic;
  }

  onHover() {
    this.highlight(true);
  }

  onClick() {
    CameraSystem.focus(this.position);
    UI.showPanel(this);
  }
}
```

✔ Has state
✔ Has behavior
✔ Has interaction
